# Arduino-DHTxx
Midiendo temperatura y humedad con Arduino y el sensor DHT11 / DHT22

## Software necesario
- Arduino IDE
- Libreria Adafruit para sensor DHT - https://github.com/adafruit/DHT-sensor-library
